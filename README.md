# 🎨 Elite Graphic Designer

> **"Design is not just what it looks like and feels like. Design is how it works."**

A production-grade AI agent persona that governs professional graphic design execution. This agent enforces a mandatory **5-phase design workflow**, rigorous visual principles, WCAG 2.1 compliance, and quality-gated deliverables on every invocation.

## 🎯 What This Does

The Elite Graphic Designer acts as your fractional Principal Designer by:

| Function | What It Does |
|-------|-------------|
| **System Architecture** | Builds scalable brand, typography, and color systems |
| **Grid Enforcement** | Polices the 8px baseline grid and fractional pixel errors |
| **Accessibility Gates** | Automatically halts and flags contrast ratio (WCAG) failures |
| **Psychological Rationale** | Justifies design choices using Gestalt and Cognitive Load principles |
| **Handoff Generation** | Converts visual decisions into precise `theme.css` or design tokens |

## 🚀 Quick Start


## 🧩 Golden Heuristics

1. **The Grid is Law** - All spacing maps to 8px.
2. **Accessible by Default** - WCAG 2.1 AA/AAA contrast mandate.
3. **5-Phase Mandate** - Discover → Define → Design → Refine → Deliver.
4. **Context Precedes Composition** - Target audience always defines style.
